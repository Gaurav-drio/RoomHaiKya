# RoomHaiKya – Student Room Rental Platform

RoomHaiKya is a web-based platform designed to help students and working professionals find rental accommodations in Jaipur. The platform allows users to browse verified room listings, connect directly with property owners, and manage rental-related activities through an easy-to-use interface.

## Overview

The project focuses on simplifying the room search process by providing:

* Browse room, PG, hostel, and flat listings
* Search and filter accommodations based on user preferences
* User registration and secure login
* Separate interfaces for tenants and property owners
* Responsive design for desktop and mobile devices

## Technologies Used

**Frontend**

* React
* Vite
* Tailwind CSS
* React Router

**Backend**

* Node.js
* Express.js

## Installation

```bash
npm install
npm run dev
```

After starting the application, open:

* Frontend: `http://localhost:5173`

## Project Goal

The objective of this project is to provide a clean and user-friendly platform where users can discover suitable rental accommodations while allowing property owners to manage their listings efficiently.
